Receipt Print Agent

Install this once on each till's computer to print receipts directly to your thermal printer — no blank gaps, no page-size settings, ever again.

Download for Windows
Windows 10/11, 64-bit. Free, installs in a few seconds.
What happens when you install it
  1. Run the downloaded file and click Yes when Windows asks for permission.
  2. It starts automatically and opens a page to pick your printer from a dropdown.
  3. Choose your printer, save — that's it. It also starts quietly every time this computer turns on, so you never have to think about it again.

Don't have a thermal printer set up yet, or skipping this for now? No problem — receipts will still print fine through your browser as they do today.
